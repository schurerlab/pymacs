# Example2: 9UWJ / AVPR1A with A1E and CLR

## Purpose

Ligand/cofactor-aware PyMACS workflow demonstrating chain selection, ligand selection, and cholesterol/cofactor handling.

## Folder layout

- `input/`: original structure inputs.
- `parameters/`: ligand/cofactor parameter seeds and prebuilt GROMACS parameter files when available.
- `prepared_system/`: curated setup outputs needed to inspect or restart the workflow.
- `completed_run/`: curated trajectory/output/analysis files from a completed example run.

Repository-level scripts, force fields, environments, and docs are intentionally not duplicated here.
