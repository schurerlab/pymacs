# Output Gallery

This repository currently includes one reference figurebook PDF and a logo image, but not a large standalone gallery of exported analysis PNGs. This document therefore mixes currently available example files with the expected outputs generated by the pipeline.

## Files currently present in the repository

- `Example/MD_ANALYSIS_FIGUREBOOK.pdf`
- `images/Pymacs_Logo.png`

## Example figurebook

- `Example/MD_ANALYSIS_FIGUREBOOK.pdf`
  - Purpose: reference example for the final PDF report generated by `4PDF4MD.py`
  - How to reproduce: run the walkthrough in `docs/USE_CASE_WALKTHROUGH.md`

## Expected analysis outputs from Step 3A

Depending on workflow mode, the analysis stage can generate figures such as:

- Protein RMSD
- Protein RMSF
- Radius of gyration (Rg)
- Ligand RMSD and ligand RMSF
- Binding-pocket contact heatmaps
- Residue contact-frequency bar plots
- Interaction composition plots
- Distance distribution plots
- NETWORX-ready network panels

Common filenames referenced by the legacy and current workflows include:

- `Protein_RMSD.png`
- `RMSD_<PROTEIN_NAME>.png`
- `RMSF_<PROTEIN_NAME>.png`
- `RMSF_<PROTEIN_NAME>_per_residue_contacts.png`
- `Radius_of_Gyration_Overlay.png`
- `binding_pocket_only.pdb`
- `binding_pocket_only.xtc`
- `<LIGAND>_contact_map_residue_time.png`
- `<LIGAND>_contact_frequency_filtered.png`
- `interaction_grouped_normalized.png`
- `interaction_stacked_normalized.png`

## Expected outputs from Step 3B

The NETWORX renderer can produce figures such as:

- `<LIGAND>_bar.png`
- `<LIGAND>_network.png`
- `<LIGAND>_expanded.png`
- combined PDF panels from the network stage

## How to build a richer gallery

1. Run the example workflow in `docs/USE_CASE_WALKTHROUGH.md`.
2. Save representative PNG outputs from `3A_AutomateGromacs.py` and `3B_NETWORX.py`.
3. Add those images to `images/` or a dedicated `docs/gallery/` folder in a future documentation update.
